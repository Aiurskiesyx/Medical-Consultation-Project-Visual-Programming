# Medical-Consultation-Project-Visual-Programming

The Clinic Consultation System is a desktop-based application developed as a Visual Programming project using the Visual Studio development environment. This application is designed to digitize and streamline the clinic's administrative workflow, from patient registration and doctor schedule searching to recording medical consultation histories.

## 🚀 Key Features

- **Patient Data Management:** New patient registration, data updates, and real-time patient profile searching.
- **Doctor Data Management:** Management of doctors' practice schedules and the medical specialties of each staff member.
- **Consultation Scheduling System:** Facilitates patients in setting up appointments with doctors on available dates and times.
- **Medical Records:** Structured recording of patient complaints, examination results, and doctor diagnoses.
- **Intuitive Desktop UI:** A user-friendly visual interface designed to make it easier for administrative staff to operate the system.

## 🛠️ Technologies Used

- **IDE:** Visual Studio
- **GUI Framework:** Windows Forms (WinForms) / WPF
- **Programming Language:** C# / VB.NET
- **Database:** MySQL (XAMPP)

## 💻 Installation and Usage Guide

1. **Prerequisites:**
   - Ensure **Visual Studio** (version 2019 or 2022 recommended) is installed with the `.NET desktop development` workload.
   - Ensure your database server (e.g., XAMPP for MySQL) is active.
   
2. **Database Setup:**
   - Open your database management tool (such as phpMyAdmin).
   - Create a new database.
   - Import the project's `.sql` database file available in this repository's folder.

3. **Running the Application in Visual Studio:**
   - Clone this repository to your local computer directory:
     ```bash
     git clone [https://github.com/your-username/clinic-consultation-app.git](https://github.com/your-username/clinic-consultation-app.git)
     ```
   - Open Visual Studio, select **Open a project or solution**, and navigate to the *Solution* (`.sln`) file of this project.
   - If necessary, adjust the *Connection String* in the source code (for example, in the `App.config` file or the database connection file) to match your local database configuration.
   - Press the **Start** button or `F5` in Visual Studio to compile and run the application.
